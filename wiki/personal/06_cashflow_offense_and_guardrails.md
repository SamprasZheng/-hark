# 現金流進攻 + 護欄

> 你的首要目標:**賺更多現金流 / 紀律化交易 → 滾錢還債**。
> 顧問鐵則:**「交易賺錢滾錢還債」正是 2025 爆倉的劇本。** 進攻保留,但被護欄包住。

## §1. 🎯 主原則(再強調)

> **房屋、還債、稅,完全靠 [[05_equity_monetization_schedule]] 的固定排程成立,不需要進攻成功。**
> 只要這三件事鎖定,Alpha 賺到 = **加速器**,賠掉 = **局部傷害**。

## §2. 進攻槓桿(合法、可持續的「賺更多」)

| 槓桿 | 內容 | 註記 |
|---|---|---|
| **NVIDIA 績效** | 明年績效 → 加薪 + RSU refresh + 升遷 | 你已在努力。**列為 upside,不寫進必須**(不確定性高) |
| **ESPP 鎖價紅利** | 2027/2 前成本 ~$97(對市價 ~$212 = 巨大折價);之後 ~$140 | **2026/8 + 2027/2 兩個 $97 窗口最肥** → 在 750萬稅線內盡量變現 |
| **海外收入 > 100萬** | 計入基本所得額 | 仍遠低於 750萬 → 申報即可,見 [[04_long_range_tax_plan]] |
| 兼職 / 副業 | 你工作高壓,難穩定增加 | **不依賴**;有餘力再說 |
| **紀律化交易(Alpha)** | 在 $hark rubric 下操作 | **受 §3 硬上限約束** |

## §3. 🔒 Alpha sleeve 護欄(更硬)

**在無擔保信貸明顯下降前**:

1. **規模上限**:Alpha sleeve ≤ **流動淨資產的 5%**(且須是「賠光也不影響房/債/稅/緊急金」的錢)。
   - 流動淨資產 = 可自由投資部位,**排除** SGOV 房屋戶、排除已被「強制變現」鎖定用途的錢。
   - 現階段債高 → 這個數字本來就很小(主要是把 P1 既有投機/槓桿部位**縮減**到上限內)。
2. **禁用工具**:**禁止保證金、選擇權裸倉、加密貨幣槓桿。**(這三項是 2025 的成因。)
3. **停損制(不是單筆停損)**:
   - **月虧損上限**:Alpha sleeve 單月虧損達 X% → 該月停手。
   - **年虧損上限**:全年累計虧損達 Y% → 全年停手。
   - **觸發後冷卻 30 天**:停手期間不開新倉,只複盤。
   - (X / Y 由你設定一個「絕對能接受」的數字,寫進此處。)
4. **資金來源**:只能用 [[05_equity_monetization_schedule]] §4 瀑布的**第 5 順位剩餘**,不得動用前 4 順位(稅/現金流/房/債)。

> 護欄的意義:進攻失敗時,最壞情況**有界**——絕不會再拖垮整體。

## §4. 月現金流 KPI 階梯(止血 → 打平 → 轉正)

目標:從「靠賣股補缺口」推進到「淨薪水即可打平」。最大槓桿是**砍掉債務月付**。

| 里程碑 | 月現金流 | 觸發條件 |
|---|---|---|
| M0 現況 | **約 −22K ~ −27K**(靠賣股補) | — |
| M1 止小血 | ~ −15K | 處理閒置機車(−2,500)+ 生活費盤點優化 |
| M2 砍一筆債 | ~ −5K ~ 0 | 清掉一筆信貸(如國泰 −12,000/月) |
| M3 薪資打平 | **≥ 0** | 三貸顯著下降 + 認列收入提升 |
| M4 轉正 | **> 0** | 信貸接近歸零,為 2030 房貸騰出空間 |

**注意時間敵人**:2026 還多一筆所得稅分期 ~16,355/月(到年底)。M2 前現金流仍緊,**這段最依賴變現排程**——但要用排程(確定性),不是用 Alpha(運氣)。
