# 長線稅務規劃(2026–2030)

> **修正**:舊頁用的「670萬」已過時。現行用 **基本所得額免稅額 NT$750萬**。
> 本頁不取代會計師。複雜案件(RSU/ESPP 來源認定、AMT)請以台灣會計師正式試算為準。

## §1. 兩條稅務軌道(務必分清)

### 軌道 1:綜合所得稅(國內薪資等)

- 你 2025 年度(2026 申報):綜所淨額 NT$2,142,950 → 20% 級距、累進差額 147,700 → **應納 280,890**;已扣繳 84,505 → **自繳 196,260**(已辦 **12 期 0 利率**分期)。
- **級距爬升風險**:綜所淨額逼近 NT$472萬+ 進入 **40%** 邊際。薪資成長 + 認列增加會推高級距。

### 軌道 2:海外所得 / 最低稅負制(AMT,基本所得額)

- **免稅額**:基本所得額 **NT$750萬**(超出部分 ×20%,與一般所得稅取**高者**繳)。
- **計入門檻**:同一申報戶**海外所得 ≥ NT$100萬**才需計入基本所得額。
- **重點**:**已實現海外所得,即使沒匯回台灣,也要申報。**〔財政部 AMT 說明:https://www.mof.gov.tw/eng/singlehtml/f48d641f159a4866b1d31c0916fbcc71?cntId=9a232ec49cf34ddfa097944223f50bda〕

## §2. ⚠ 關鍵未定:RSU/ESPP 算「國內薪資」還是「海外所得」?

這會大幅改變你的稅,**必須請會計師認定**:

- 若 NVIDIA 台灣以**台灣薪資扣繳**(開扣繳憑單)→ 進**軌道 1 綜所稅**。
- 若由 **NVIDIA 美國**支付、非台灣薪資 → 進**軌道 2 海外薪資所得**(AMT)。

舊 [[../12_employee_concentration]] 假設 RSU/ESPP vest = 海外薪資所得。**請於 8 月前找會計師確認你的實際情況**,後續 ledger 才能歸對軌道。

## §3. 各類股權的稅性(美國端 + 台灣端)

| 項目 | 美國端 | 台灣端 |
|---|---|---|
| RSU vest | NVDA 預扣 22–37% 給 IRS(以股扣繳) | 薪資所得(軌道待 §2 認定) |
| ESPP 折扣部分 | 折扣視為 ordinary income,已預扣 | 薪資所得 |
| **賣股 capital gain** | 非 US resident **無美國資本利得稅** | **海外財產交易所得**(軌道 2,計入 AMT) |
| 股息 | 預扣 30%(無法避免) | 海外所得 |

**推論**:
- **RSU vest 當天賣** → 賣價 ≈ vest FMV → capital gain ≈ 0 → 幾乎不增加海外所得。這是 [[05_equity_monetization_schedule]] 要求「即賣」的稅務理由。
- **ESPP(成本 ~$97 vs 市價 ~$212)** → 賣出實現大額 gain → **海外所得增加最多**,8 月 / 明年 2 月窗口要特別盯 750萬線。

## §4. 逐筆賣股 ledger(強制 · 全年滾動)

> **不要在年底才回頭算。** 每賣一筆就登錄,隨時看到「距 750萬 還剩多少」。與 [[05_equity_monetization_schedule]] 同步。

| 成交日 | 交割日 | 標的 | 股數 | 賣價(USD) | 成本基礎(USD) | 匯率 | 台幣海外所得 | 年度累計海外所得 | 距 750萬餘額 | wash-sale 風險 | 用途 |
|---|---|---|---|---|---|---|---|---|---|---|---|
| TBD | | | | | | | | | | | 房/債/現金流/換股 |

**規則**:
- 年度累計海外所得逼近 **NT$750萬** → **暫緩當年剩餘賣出**,推遲到次年。
- 賣出登錄當下即估算「台幣海外所得 = (賣價 − 成本) × 股數 × 匯率」。

## §5. 長線稅務策略

1. **分年實現利得**:大筆已 vested(~$130K)分 2–3 年賣,避免單年衝破 750萬觸發 AMT。
2. **vest 當天賣**:把 capital gain 壓到 ≈ 0(同 §3)。
3. **Loss-harvest**:賣出美股已實現虧損(P1 投機部位)抵減海外所得;**注意 wash-sale 30 天**(賣後 30 天內不可買回同標的)。待 [[01_financial_profile]] §6 補虧損金額。
4. **年底總檢**:每年 11–12 月檢視全年海外所得,接近門檻則暫停賣出。
5. **報稅季**:每年 3–5 月跑年度試算 + AMT 檢查(顧問節奏)。

## §7. 節稅策略(綜所稅:免稅額 / 扣除額)— 你可能正在多繳稅

### 扶養親屬免稅額(每名 +NT$97,000)

- **媽媽(癌症,你給孝親費 2萬/月)**:若她所得低、確由你扶養 → 可列**受扶養親屬 +97,000 免稅額**;且她在健保特約醫療院所的**醫藥費可「列舉扣除,無金額上限」**。
  - ⚠ **這可能是你目前最大、最常被漏掉的節稅。** 請確認報稅時(a) 是否已把媽媽列為扶養、(b) 是否申報她的癌症醫藥費。若沒有,過去年度可能可更正申報。
- **未來每名子女 +97,000 免稅額**。

### 子女相關(規劃 1–2,上限 3)

- **幼兒學前特別扣除額**:近年放寬(第一名約 15萬、第二名起約 22.5萬/年,並取消排富),幼兒期適用。
- **教育學費特別扣除額**:子女就讀大專以上,每人 NT$25,000/年(較後期)。
- 免稅額 + 扣除額**隨子女數疊加** → 1/2/3 個孩子情境差異大,壓測見 [[07_stress_tests]]。

### 房屋節稅

- **現在(租屋)**:房租支出**列舉扣除**,上限約 NT$180,000/年(需自住、未在他處有屋等)。⚠ 確認你是否已在列舉。
- **交屋後(~2030)**:自用住宅**購屋借款利息扣除**,上限 NT$300,000/年(需減除當年度申報的儲蓄投資特別扣除額)。見 [[03_house_funding_plan]]。

### 標準 vs 列舉

- 有大額醫藥費(媽媽)+ 房租 / 房貸利息時,**列舉扣除**通常較有利;**每年比較**標準扣除 vs 列舉,擇高。

## §8. 股息 / 存股策略(先更正,再用對)

### ⚠ 更正:中華民國現行法**沒有「30萬股息免稅」**

- **NT$270,000(27萬)是「儲蓄投資特別扣除額」**,只適用**金融機構存款利息**(及部分債券利息)→ **不含股利、不含郵政存簿利息**。你記的「30萬股息免稅」是把這條和股利混在一起了。
- **股利所得**走兩制擇優:
  - ① **併入綜所**,給 **8.5% 可抵減稅額,每戶上限 NT$80,000/年**(約 94萬股利打滿);
  - ② **分開計稅 28%**。
- **你屬 20%(可能 40%)級距**:小額股利用①(邊際 − 8.5%);股利很大、到 40% 時②(28%)可能較省。**對你而言國內股利不是「免稅」,只是打 8.5% 折。**
- **海外股利(NVDA 等)**:不適用 8.5% 抵減,屬**海外所得(AMT)**,美國預扣 30%。
- **二代健保補充保費**:單筆股利 ≥ NT$20,000 課 **2.11%**。

### 存股 / 零股的正確定位(對你)

- 「領股息免稅」對**高所得的你效果有限**。你最大的稅槓桿其實是 §7 的**扶養(媽媽+小孩)+ 房貸利息**,以及**海外所得 AMT 時點管理**——不是國內股息。
- 但**建立國內台股高股息 sleeve(你 9A92 已有 0056/00878/00929/00965/00983A)仍有價值**:**被動 NTD 現金流**,不必賣 NVDA(省 FX + 降集中度),且享 8.5% 抵減 → 直接服務你的首要目標(現金流)。
- ⚠ **順序鐵則**:**在高利率信貸還清前,先還債、再存股。** 一邊背 ~高利信貸、一邊買殖利率 5–6% 高股息 = **負利差,數學上輸**。存股 sleeve 是「債務下降後」的現金流引擎,資金走 [[05_equity_monetization_schedule]] §4 **第 5 順位剩餘**,排在 [[02_debt_and_consolidation]] 之後。
