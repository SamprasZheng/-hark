# 壓力測試(5 情境)

> 每個情境必答 4 題:**① 房款付不付得出?② 債務會不會違約?③ 要賣多少股?④ Alpha sleeve 是否歸零?**
> 任一情境讓房/債/稅斷裂 → 回頭調 [[02_debt_and_consolidation]] / [[03_house_funding_plan]] / [[05_equity_monetization_schedule]] 的排程,直到全綠。
> (含 `TBD` 之精確數字待 [[01_financial_profile]] 補齊;此處先建立框架與定性判讀。)

## 情境 1:NVDA −50%(例:$212 → $106)

| 問 | 判讀 |
|---|---|
| 房款 | **若頭期已撥入 SGOV → 不受影響(PASS)**。未來期款靠未來 vest 者:同金額需賣 **~2 倍股數** |
| 債務違約 | 底薪不受股價影響;但 RSU/ESPP 變現能力腰斬 → 補現金流能力下降。**信貸已降低則安全,仍高則吃緊** |
| 賣多少股 | 達同樣台幣需賣約 2 倍股數(但 capital gain 小 → 稅輕) |
| Alpha 歸零? | 規則上 Alpha 早已 ≤5% 並設停損;−50% 環境極可能觸發停損 → **冷卻 30 天** |

**結論**:SGOV 鎖定的房款是這情境的命門。**越早撥足 SGOV、越早降債,−50% 越扛得住。**

## 情境 2:失業 6 個月

| 問 | 判讀 |
|---|---|
| 房款 | SGOV 已鎖部分安全;未來期款若靠「未來 vest」→ **中斷** |
| 債務違約 | 月付(三貸+車貸 63K)+ 房租等 → 6 個月燒 **~80–90萬**。**需緊急預備金 + 賣已 vested** |
| 賣多少股 | 賣已 vested(~$130K)的一部分撐 6 個月 + 房款;**離職後未 vested 多半作廢(forfeit)** |
| Alpha 歸零? | **立即歸零** |

**結論**:這是最危險情境之一。**緊急預備金(目前 TBD,關鍵缺口)** = 至少 6 個月固定支出(~6 × 13萬 ≈ **78萬**)。**先把緊急金補起來**,見 [[08_insurance_and_family_risk]]。

## 情境 3:2030 房貸利率 3.5–4%

| 問 | 判讀 |
|---|---|
| 房款 | 頭期已鎖 → 付得出(PASS) |
| 債務違約 | 月付升到 **5.9–6.7萬**。**若 2030 前清掉舊信貸 → 只背房貸,認列收入 ~15–17萬可撐**;**舊信貸沒清 → 5.4萬+6.7萬=12萬月付 → 崩** |
| 賣多少股 | 不影響頭期;影響長期月付承受力 |
| Alpha 歸零? | 不直接相關 |

**結論**:**PASS 的前提 = 2030 前舊信貸歸零 + 認列收入提升。** 這就是 [[03_house_funding_plan]] §4 硬目標的存在理由。

## 情境 4:RSU refresh 低於預期

| 問 | 判讀 |
|---|---|
| 房款 | 已鎖部分安全;未來期款需更高比例變現或拉長時程 |
| 債務違約 | 還債速度放慢,但不立即違約 |
| 賣多少股 | 需賣更高比例既有持倉補足計畫 |
| Alpha 歸零? | 不直接 |

**結論**:**計畫基準必須用保守 refresh(持平或略降),不能假設成長。** 符合主原則「不靠進攻」。refresh 超預期 = upside,不是前提。

## 情境 5:家庭醫療支出 +20萬/年(媽媽治療升級)

| 問 | 判讀 |
|---|---|
| 房款 | 間接擠壓(資金被醫療分走) |
| 債務違約 | 現金流缺口再 **+~1.67萬/月** → 還債放慢 |
| 賣多少股 | +20萬/年 → 多賣對應股數 |
| Alpha 歸零? | **醫療優先,Alpha 讓位** |

**結論**:現金流要**預留醫療緩衝**,並補足**重大傷病 / 實支實付醫療**保險,見 [[08_insurance_and_family_risk]]。

## ⚠ 複合最壞情境:NVDA −50% + 失業(同時)

AI 循環反轉常伴隨科技業裁員 → 收入與資產**同時崩**(三倍打擊)。這情境下唯一的防線是:
1. **SGOV 鎖定的房款**(切斷與股價的相關性)
2. **6 個月緊急預備金**
3. **已大幅下降的信貸**

→ 三者都是**現在就要動工**的事,不能等。
