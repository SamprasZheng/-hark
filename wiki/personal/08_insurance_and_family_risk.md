# 保險與家庭風險

> 你已有部分保單 → 本頁做**現有保障盤點 + 補缺口**,不推銷新儲蓄險。
> 核心原則:**你最貴的問題是高月付債務 + 流動性。** 保費花在**純保障**,不要把現金鎖進低報酬保單。

## §1. 第一優先:緊急預備金(不是保險,但更急)

[[07_stress_tests]] 情境 2(失業 6 個月)顯示這是最大缺口。

- **目標**:至少 **6 個月固定支出 ≈ 6 × 13萬 ≈ NT$780K**(現金 / 台幣活存 / 短天期定存)。
- **現況**:TBD(請補手上台幣現金緩衝)。
- **與還債的取捨**:在「先建緊急金」與「先還高利率債」之間 → 建議**先建至少 3 個月(~40萬)的最低緩衝**,再全力還債;不要 0 緩衝就猛還債(否則一個意外又得借高利)。

## §2. 純保障優先序(你的處境)

| 險種 | 為什麼對你重要 | 註記 |
|---|---|---|
| **定期壽險(Term)** | 你撐著孝親費 + 未來配偶/子女 + 2030 起 1400萬房貸。**你倒下,債務與家庭誰扛?** | 保費低、保額高;**交屋後保額應涵蓋房貸餘額 + 信貸 + 家庭數年支出**。可用遞減型 / 房貸壽險對應房貸餘額 |
| **失能扶助險** | 高壓工作;**喪失工作能力 = 收入停、債務在** | 比壽險更常被忽略,但對「還在背債的人」極關鍵 |
| **重大傷病險** | 家族癌症史;確診一次給付一筆 | 自身罹病時的現金緩衝 |
| **實支實付醫療** | 補健保不給付的住院/手術/材料缺口 | 檢查現有醫療險的實支額度與正副本 |
| ~~儲蓄險 / 投資型保單~~ | **避免** | 鎖住流動性、報酬低;你還 20% 高利債,遠勝買 2% 儲蓄險 |

## §4. 家庭與人生事件風險(時間軸見 [[01_financial_profile]] §4)

| 事件 | 風險 / 需求 | 規劃方向 |
|---|---|---|
| **媽媽癌症(進行中)** | 孝親費 2萬/月 + 醫療可能升級(壓測情境 5:+20萬/年) | 現金流預留醫療緩衝;媽媽既有保單盤點(新保多半受限) |
| **結婚** | 一次性支出 + 家庭財務合併 | 配偶若有收入 = **雙薪**改善現金流;婚前財務透明 |
| **生子(1–2,上限 3)** | 生產 + 月子 + 育兒;費用隨人數疊加 | 與房款期程錯開;預留專款;**壽險保額隨子女數 + 房貸調升** |
| **子女教育** | 長期、金額大 | 待房貸/信貸穩定後,另開教育金規劃(屆時可考慮低成本指數,不買教育儲蓄險) |
| **父母健康** | 醫療 + 照護 | 緊急金 + 重大傷病(自身)+ 預留孝親緩衝 |

## §5. 與整體計畫的關係

- 緊急金(§1)是 [[07_stress_tests]] 情境 2 的命門。
- 純保障(§2)讓「家庭風險」不會逼你去借高利債 → 保護 [[02_debt_and_consolidation]] 的還債進度。
- 人生事件(§4)的資金需求,需排進 [[05_equity_monetization_schedule]] 的長期變現規劃,**且不得佔用房/債/稅的固定排程**。
