# DOT 持倉檢討 + 往後立場（postmortem）

> 一次性檢討你過去在 Polkadot/DOT 的重虧 + 往後該怎麼站。**誠實、反 hype**——不為了安慰你而看多。

## 0. 一個數字

**DOT −97.83% from ATH**（$54.98 @ 2021-11-04 → **$1.20** @ 2026-05-30）。市值 $2.02B、**排名跌到 #44**。
近 1 年 −70.8%、近 30 天 −1.35%（在底部陰跌，不是在反彈）。這不是「回檔」，是**結構性毀滅**。

## 1. 當年大概錯在哪（四個結構性原因，非運氣）

1. **敘事衰退 + 開發者心智份額流失（最致命）。** Polkadot 的「parachain / 共享安全 / Layer-0」敘事在 2021 是熱點，
   但隨後**輸給 ETH L2、Solana、Cosmos**。live 佐證（The Defiant 2026-05，見 [[../raw/kol_signals/crypto-kol-profiles-2026-05-31]] Gavin Wood）：
   Polkadot DeFi **TVL ~$81M**，對比 **ETH ~$48B / SOL ~$6.8B**——差 2-3 個數量級；**月活 230K（2024-01）→ 43K（2026-05）**；
   旗艦 Hydration −80%；治理介面 Polkassembly 關閉，理由是「ecosystem leadership 缺乏方向」。
2. **稀釋 / 通膨（現在才補，太晚）。** 上一輪 DOT 年通膨 ~10%、無上限，持有者長期被稀釋。**2026-03 硬上限**（見下）
   才把發行砍 53.6%——但這是在價格已 −98%、生態已空之後。
3. **價值捕獲弱。** DOT 的費用 / 真實使用量極小，token 與「網路有沒有被用」的連結薄弱；質押收益來自增發（左手換右手），
   不是來自網路現金流。
4. **結構性跑輸——連同類都輸。** 不只輸 BTC（這輪 BTC 才 −41%），也輸多數其他 L1。**「願景以白皮書出貨，比使用量和價格快」**
   （Wood 的 JAM/Gray Paper 一再推出宏大架構，**JAM mainnet 已跳票到 2027**）。

## 2. 2026-03 硬上限催化劑 — 誠實解讀

- **事實**（[[../../yxz/wiki/concepts/dot-hard-cap]]，已查證）：2026-03-14 Referendum 1710、81% 通過，**總量封頂 2.1B**，
  **年發行砍 53.6%**，驗證者獎勵上限 $5k/月、隔年減半。
- **live 對帳**：snapshot `max_supply = 2.1B`、`circulating = 1.686B`（**80.3% 已發**）→ 硬上限已生效，剩 ~20% 待發。
- **誠實判讀**：**「降低賣壓 ≠ 有人想買」。** 硬上限解決的是「**被多稀釋**」，不是「**需求回來**」。把放在沒人用的 token 上的
  通膨關掉，不會讓它漲——只是讓它**少跌一點**。這是「減少傷害」的催化劑，**不是逢低加碼的理由**。

## 3. DOT 現在在哪

- 排名 #44、市值 $2.02B；曾經 top-5/10 的 Layer-0 敘事旗艦，如今是中型幣。
- 生態指標（TVL/MAU/旗艦協議/治理）全線收縮；JAM 是技術上可信、但**採用與價格的指引極差**的多年期 R&D。
- 一句話：**技術可信度高、但作為投資標的，敘事與使用量的證據與「再起」相反。**

## 5. 誠實的底線

- DOT 讓你重虧的**那個論點（敘事 + 生態起飛）結構性壞了**。硬上限是「少被稀釋」，不是「需求回來」。
- **不要為了攤平而攤平。** 沉沒成本 + 一個減稀釋催化劑，正是會誘你再進場的組合——而那是當年毀滅的同一種心理。
- 若你仍信 JAM 的長線技術賭注，**把它當一張極小的彩票**（在 4% 硬頂內、現貨、不加碼），而不是「翻身倉」。
- **你的翻身不在 DOT。** 在 NVDA 薪酬 + 還債 + 紀律分散。crypto（含 DOT）是小而冷的衛星，不是引擎。
