# 底層邏輯:regime 決定廣度 → 大浪需要「燃料」(2026-06)

> 主理人(2026-06):現在**不是 2021 大降息瘋牛**,覆蓋範圍(廣度)絕對比較小。
> 可能只能挑「**真的會賺錢的(科技)股**」或「**真的有題材的股**」,才有機會展開**大浪型**。

這條被固化進 `rally_signal.py` 的評分(`tight_regime` + `has_fuel`)。

## 為什麼對(流動性決定廣度)
- **2021 = loose regime**:零利率 + 大放水 → 流動性氾濫 → **雞犬升天**,連沒盈利的殼都能噴。
  廣度大,亂買也能賺(覆蓋範圍廣)。
- **2026 = tight regime**:利率不低、沒有大放水 → 流動性有限 → 華爾街**沒有足夠資金把所有
  錯殺股都炒起來**。廣度小,資金只會集中在**少數有「燃料」的票**。
- ⇒ 「行情未完、輪動到錯殺股」**成立,但條件變嚴**:只有有燃料的會展開大浪,其餘只是反彈。

## 「燃料」= 真會賺錢 或 真有題材
- **真會賺錢**:基本面(FOM quality)夠強 → 有盈利支撐的上漲才持久。
- **真有題材**:供應鏈 design-win / 消息催化 → 像 INTC/MU 的轉機、SpaceX 的太空題材。
- 兩者皆無 → 就算技術+資金讓它彈,**這個 regime 撐不起大浪** → 標 `🪨 缺燃料·反彈非大浪`,不追。

## 在系統裡怎麼運作(`rally_signal`)
- `has_fuel` = 基本面 / 供應鏈 / 消息 任一 ≥ `WAVE_FUEL_MIN`(55,比 catalyst 的 50 嚴)。
- `tight_regime`(預設 **True**)下:
  - **買入考慮**(`buy_consider`)一定要 `has_fuel`(不只是 catalyst)。
  - 有動但無燃料 → 降級 `🪨 缺燃料·反彈非大浪`,不給買訊。
  - `wave_candidate` = 有燃料 + 起漲 = 夠格展開大浪。
- `loose`(2021 式)可放寬回 catalyst 級(模擬雞犬升天)。
- 切換:`assess(..., tight_regime=False)` 或 `build_signals(..., tight_regime=False)`。

## 操作含義(收斂選股)
1. **科技股**:只挑**真會賺錢**的(高 quality),不是所有 AI 錯殺股都買。
2. **題材股**:只挑**真有題材**的(供應鏈/消息實證),純敘事的打 🚫 墓園型。
3. **錯殺反彈但沒燃料**(很多深跌民生/小型):這個 regime **只能反彈、難成大浪** → 不追、或極短打。
4. 仍受 regime gate 第一節制:資金面翻 STRESS → 連有燃料的高 beta 也先減。

## 一句話
**2021 是「亂買都漲」;2026 是「只有真賺錢或真題材的才漲大浪」。** 系統用 `has_fuel`
把這條紀律自動套在每一檔上。
