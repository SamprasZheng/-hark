# Sivers Semiconductors(STO:SIVE / OTC:SIVEF)

瑞典光子+毫米波半導體微型股(~$130M mcap,grade C 來源)。兩個事業段:
**Photonics**(DFB 雷射陣列、光晶片)與 **Wireless**(mmWave 5G/SATCOM beamformer)。
P1 自 2026-06 起持有(OTC F 股 SIVEF;見 [[../../wiki/positions]])。

## 論點:CPO 的外部光源(ELS)咽喉點

Principal 論點(2026-06-12):「Big choke points for CPO」。機制:co-packaged optics
把光學引擎搬進 switch/XPU 封裝,但 **DFB 雷射對溫度敏感、不能貼著高熱 ASIC** →
雷射必須外置成 External Light Source(ELS)模組 → ELS/雷射陣列是 CPO 架構的必經節點。
Sivers 卡位:

- 2026-03:與 O-Net(ODM)+ Enablence(NxN star coupler)合作 ELS 模組,目標 AI
  資料中心 CPO 部署。來源:[PR Newswire](https://www.prnewswire.com/news-releases/sivers-semiconductors-o-net-and-enablence-technologies-announce-external-light-sources-for-ai-datacenters-302715690.html)、
  [Semiconductor Today](https://www.semiconductor-today.com/news_items/2026/mar/sivers-onet-enablence-170326.shtml)(retrieved 2026-06-12,grade B)
- 2026-06:與 **GlobalFoundries** 合作 AI 基建矽光子;Sivers 雷射陣列進 GF 參考設計
  (CPO/LPO/互連)。來源:[PR Newswire](https://www.prnewswire.com/news-releases/sivers--globalfoundries-advance-ai-data-center-optical-solutions-302788035.html)、
  [Semiconductor Today](https://www.semiconductor-today.com/news_items/2026/jun/SIVERS-GF-020626.shtml)(retrieved 2026-06-12,grade B)
  — **2026-06-11 單日 +12% 的催化即此**(P1 快照 day chng 印證)。

  (二手轉述,grade C,未讀原報告)。
