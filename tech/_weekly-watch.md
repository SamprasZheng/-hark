2. Apply [[_sourcing-protocol]]: re-source any `single-source-pending` or >1-quarter-old figure; flag contradictions.
3. On a status flip (e.g. ⏳→✅), update the FOM 意涵 column and re-run FOM on the affected `TECH_DD_NODES` ([[fom-integration]] §4).
4. Emit promotions/demotions to a report for **Risk-Officer review — never auto-act**.
5. Append a dated line to §3 (changelog); if a page's verdict changes, also note it in [[../wiki/log]].
