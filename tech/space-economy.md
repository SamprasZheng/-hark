# 太空經濟 — 發射成本崩塌是真質變、在軌經濟仍太早 / Space Economy: The Launch-Cost Collapse Is the Real 質變, the In-Space Economy Is Still Too Early

## 0. 一句話判決 + Desk View
**結構 (structural, conf 0.74)** — 但必須把「發射」與「在軌經濟」嚴格切開,它們是兩個不同成熟度的故事。**發射 = 現在進行式的真質變**:SpaceX 2025 年單以 Falcon 9 飛了 **165 次**(占全球 324 次的 ~51%、發射全球 85% 的衛星),B1067 助推器達成史上第 31 次重複使用;Falcon 9 真實邊際成本已被估到 ~$300/lb(~$1,500/kg 量級,約客戶報價的 25%)[BryceTech via Via Satellite — satellitetoday.com — retrieved 2026-05-31 — grade B — cross-confirmed][NextBigFuture Falcon true cost — nextbigfuture.com — retrieved 2026-05-31 — grade C — single-source-pending]。可重複使用 + 高頻次把 $/kg 壓低一個數量級,這就是解鎖下游一切的 enabling 質變。**在軌經濟(ISAM / 太空製造 / 軌道資料中心)= 真實但太早**:故事正在從「科學實驗」變成「有合約」(Varda 簽下 United Therapeutics 製藥、Starcloud 在軌跑 H100),但量級仍小兩三個數量級。Desk view(謹慎):**最乾淨的 listed 純標的是 Rocket Lab(RKLB)**——唯一同時擁有發射 + 太空系統的上市端到端玩家,Q1-2026 營收 $200.3M(+63.5% YoY)、backlog $2.2B,但市值 ~$72B = ~90x P/S,Neutron 首飛是把整個估值押上去的二元賭注 [RKLB Q1-2026 — investing.com — retrieved 2026-05-31 — grade B — cross-confirmed]。LUNR/RDW/PL 不是「沒營收的敘事」——三者營收都在加速——而是「估值已先行透支真實但早期的營收」。**最大盲點**:同溫層把「Starship 改變一切」直接外推到「在軌資料中心/太空工廠明年起飛」,但 Starship V3 才剛 2026-05-22 首飛、軌道加注尚未示範,而 Blue Origin 的 New Glenn 在 2026-05-28 靜態點火時整支炸在唯一的軌道發射台上——**第二供應商的脆弱性是這個 cluster 最被低估的單點風險**。

## 1. 技術底蘊 (A1) — ENGINEER-GRADE:發射成本崩塌、可重複使用、ISAM 落差
工程核心是一條鐵律:**$/kg 由「硬體攤提次數 × 翻新成本」決定,不由火箭大小決定**。
- **Falcon 9(已證明的質變)**:全推進著陸 + 整流罩回收,單一助推器 B1067 已飛 31+ 次(2025-10-19),全箭隊 55 枚助推器多次飛行;2025 年 165 次發射、僅 3 次未回收 [Falcon 9 records — Wikipedia/NASASpaceflight — retrieved 2026-05-31 — grade B — cross-confirmed]。客戶報價 ~$67M,但內部 Starlink 任務真實成本估 ~$15–30M,$/kg 已壓到 ~$1,500–3,000 級——這是**已實現、難複製**的工程護城河(翻新週期、回收率、垂直整合)。
- **Starship(質變的下一檔,尚未證明)**:目標是「整箭全重用」把 100–150 噸送 LEO。2026-05-22 V3(Block 3,Raptor 3 + 軌道加注硬體)首飛成功濺落印度洋(累計第 12 飛、7 成 5 敗),Flight 13 排程 2026-06 [Starship V3 Flight 12 — Space.com/Wikipedia — retrieved 2026-05-31 — grade B — cross-confirmed]。但**兩個未解的工程閘門**:(a) **軌道推進劑轉移**尚未示範(2026 計畫,Artemis HLS 需 ~10 次加油飛行充填深空 depot);(b) **上面級回收 catch** 尚未嘗試(只接過 booster)。$/kg 目標(Musk 喊 $10–20/kg)在高重用下分析估 ~$16–94/kg,但這是 projection,非已實現 [Starship $/kg — NextBigFuture/Netizen — retrieved 2026-05-31 — grade C — single-source-pending]。
- **ISAM / 在軌經濟(物理可行、商業太早)**:微重力製藥(Varda W 系列已 3 次返艙成功、W-4 在軌)、在軌組裝、軌道資料中心(Starcloud-1 2025-11 在軌運行 NVIDIA H100、首次太空訓練 LLM)。物理優勢真實(微重力結晶、太陽能 24/7、輻射散熱),但「把實驗變供應鏈」的單位經濟尚未閉合。

**A1=2**:發射端是已實現、難複製的硬質變(可重用 + 頻次 + 垂直整合);ISAM/ODC 物理底蘊真實但商業化未證明。發射的 moat 是**製造節奏 + 回收**,ISAM 的 moat 是**返艙/在軌操作**——兩種不同護城河。

## 4. 受益 / 受損 / 抄底 (A4)
- **結構受益(發射層,已確認)**:**SpaceX/Starlink**(現金牛 + 史上最大 IPO + Falcon 壟斷 ~51% 全球發射);上游與發射受惠者透過降本擴大 TAM。**Rocket Lab(RKLB)** 是**唯一上市的端到端純標的**——Space Systems 已占營收 68%、backlog 42%,證明它正從「小火箭公司」轉型「太空系統公司」,但 Neutron(13t LEO、報價 $50M)首飛跳票風險把 ~90x P/S 的估值全押上去。
- **方向對但估值透支(在軌/EO 層)**:**LUNR**(營收 ≈3×、首度正 Adj EBITDA,但靠 $800M 併購撐 + IDIQ 上限非 booked)、**RDW**(P/S ~10x,Zacks 2026-02 下調至 Strong Sell,均價目標較市價 -40%)、**PL**(P/S ~51x、forward ~21x vs 三年中位 3.6x,首度獲利但仍 -80% 淨利率)——三者**營收真實但估值已把多年成長定價**。
- **受損 / 單點風險(LOSERS)**:**Blue Origin**(未上市)——New Glenn 2026-05-28 靜態點火在唯一軌道發射台 LC-36 整箭爆炸、摧毀避雷塔 + 運輸豎起架,NG-4(Amazon Leo)延宕,衝擊 Blue Moon 登月與 NSSL 認證,且 BE-4 共用問題可能波及 ULA Vulcan [New Glenn 爆炸 — Spaceflight Now — retrieved 2026-05-31 — grade A — primary-fetched]。**第二/第三供應商的執行脆弱性**是 cluster 最被低估的風險:發射看似「結構」,但實際上是**單一供應商(SpaceX)壟斷**,任何挑戰者一次台架事故就回到原點。
- **抄底候選 vs 價值陷阱**:**RKLB** 是最像「真成長抄底」者——但抄底點繫於 **Neutron 首飛成功**這個二元事件(失敗或滑入 2027 = 估值重挫);分析師建議部位 1–3% 反映此二元性。**RDW** 若 de-rate 到接近歷史中位(P/S ~1–2x)且 backlog book-to-bill 維持 >1.5x,才是 value;當前 ~10x 仍偏貴。**A4=1**:節點清楚(發射降本 + 太空系統 backlog),但純標的稀少且估值透支——SpaceX 未上市(IPO 前無法參與)、RKLB 估值押在 Neutron、LUNR/RDW/PL 方向對但價格貴;最乾淨的「結構受益」(SpaceX)恰恰是散戶買不到的那個。

## 5. 多時程 (T0–T3)
- **T0(現在)結構**:發射 = **已實現質變但 priced-in**(Falcon 165 次/年、$/kg 崩落);太空系統/EO 營收加速但估值先行。
- **T1(1–3y)質變**:Starship 進入軌道級重用 + 軌道加注示範、Neutron 首飛、發射 $/kg 出現結構性 break;這是**發射端從「便宜」走向「極便宜 + 大運力」**的關鍵躍遷,真正解鎖下游。
- **T2(3–5y)太早→結構化中**:ISAM(微重力製藥商業供應鏈)、在軌組裝、首批軌道資料中心商用節點;單位經濟可能在此區間首次閉合,但仍是早期。
- **T3(5–10y)太早(現在說 = 過頭)**:軌道資料中心規模化(Starcloud-2 Blackwell、Google Suncatcher TPU)、太空工廠量產——綁定 yxz LEO/ODC wiki,但**現在把它當投資主軸 = 太早**($/kg 未到、輻射/散熱/在軌維運未證明)。**裁決:發射 結構(T0)→ 質變(T1);在軌經濟 太早(T2–T3)。**

## 8. 同溫層 + 自我打臉
**同溫層 gap**:多頭把「Starship 改變一切 + SpaceX $2T IPO」直接外推成「軌道資料中心/太空工廠明年起飛、太空股全面噴出」。數據打臉:(1) Starship V3 才剛 2026-05-22 首飛、軌道加注與上面級回收**兩道閘門都還沒過**;(2) 在軌經濟營收仍是「百萬美元級實驗」對比發射的「百億美元級 P&L」,差兩三個數量級;(3) 「太空股」估值已透支——PL ~51x P/S、RKLB ~90x P/S,RDW 被 Zacks 下調 Strong Sell;(4) 發射看似「結構」,實則 SpaceX **單一供應商壟斷**,New Glenn 一次台架爆炸就證明第二供應商有多脆弱。

**自我打臉(打我自己的「太早」空)**:我說在軌經濟「太早」,但 (a) Varda 已簽 United Therapeutics **真商業製藥合約**(非政府 demo)、3 次返艙成功;(b) Starcloud 17 個月變獨角獸、NVIDIA 親自背書、H100 已在軌跑 LLM 訓練;(c) LUNR/PL **已經做出正 Adj EBITDA / 正 FCF**——這些都比我的「實驗階段」論更早地把在軌經濟從敘事推進到合約與獲利。結論校正:**發射 = 結構(現在)→ 質變(T1);太空系統/EO = 真實加速 P&L 但估值透支;ISAM/ODC = 太早(T2–T3)但商業化拐點比預期近**。最大未知三條:Starship 軌道加注能否如期、Neutron 首飛成敗、以及 SpaceX 壟斷下「第二供應商」(Blue Origin/RKLB Neutron)能否真正建立——前者決定下游 TAM,後二者決定這個 cluster 是「壟斷」還是「市場」。
