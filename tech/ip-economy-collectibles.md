# IP / 情緒消費經濟 — 谷子·盲盒·重啟 / The IP & emotional-consumption economy

## 0. 一句話判決 + desk view / Verdict
「情緒/IP 消費」作為年輕世代的需求底層**是真的結構轉變**(綁 [[youth-culture-shifts]]),但「誰擁有持久 IP 護城河」分歧極大:Pop Mart 已從單店玩具商**質變為跨國 IP 平台**(FY2025 營收 RMB 37.12B、+184.7%;海外 43.8%)[1][2],但同一份財報暴露單一 IP(THE MONSTERS/Labubu 佔 38.1%)集中風險,且 Q4 增速放緩、二級市場退燒,股價在 2025/08 高點後一度 −25%、年報後再 −15%[3][7]。**判決:結構**(confidence 0.74),但 verdict 隨時程分裂——**T0 過熱**(估值與情緒擠泡沫)/ **T1–T2 結構**(平台基本面真實但已定價)/ **T3 質變**(若「IP 即注意力時代消費護城河」成立)。Desk view(caveated, 非投資建議):IP 平台的「重複造星能力」比任何單一角色更值錢;Pop Mart 是這波最強的造星機器但也是估值最緊的標的;真正被低估的可能是**已驗證 IP 飛輪重啟**的老牌(Disney DTC 轉正、Bandai Gundam 創高、Sanrio 授權復興),而非追逐下一隻 Labubu。

## 1. IP / 產品底蘊 / IP & product moat (A1=2)
護城河不在「某個角色紅」,而在**可重複的造星管線 + 多通路變現飛輪**。
- **Pop Mart**:盲盒(隨機性×稀缺×社群曬單)是行為機制護城河;FY2025 六個 IP 年營收破 RMB 20 億、17 個 IP 破 1 億,證明非單點[6]。Labubu(藝術家 Kasing Lung 的 THE MONSTERS)2025 銷量破 1 億隻[5]。通路=自營店+機器人商店(自販)+抽盒機 App+海外旗艦,情緒消費被工業化。
- **Bandai Namco**:「IP axis 策略」教科書級——動畫×模型(GunPla)×卡牌×街機×展覽×live 串成飛輪,Gundam FY2026 達 ¥254.3B 創歷史新高、超越 One Piece(¥139.3B)與 Dragon Ball(¥138.0B)[8][9]。
- **Sanrio**:50 年角色庫(Hello Kitty 50 週年、Kuromi/My Melody)+ 輕資產授權模式,FY3/2025 營業利益率 35.8%、ROE 48.6%[4]。
- **Disney**:樂園+影視+串流+消費品的「IP-to-everything」最完整,但近年原創/續集命中率下滑(見 §8)。
護城河強弱排序(desk view):造星可重複性 Pop Mart≈Bandai > Sanrio(吃存量)> Disney(資產重但飛輪最廣)> Hasbro/Mattel(單點電影依賴)。

## 3. 資金與權威背書 / Capital & authority (A3=2)
- **獲利自證(權威級)**:Pop Mart FY2025 淨利 ~RMB 13B、Sanrio 營益率 35.8%/ROE 48.6%、Bandai 營益 ¥189.5B、Disney DTC 轉正 +$1.3B——四家都用 GAAP 獲利+現金流證明情緒消費可規模化變現,屬 A/B 級硬證據[2][4][9][11]。
- **市值與機構**:Pop Mart 一度被 36Kr 等稱「$40B Pop Mart」,2025 股價即便回檔仍 YTD +186%[7][14];JPMorgan 點名 Twinkle Twinkle 為突破型新 IP[6]。
- **第三方權威**:Time 將 Pop Mart/盲盒列為 Gen-Z「奢侈替代」消費現象[15];CNBC、SCMP、Bloomberg 對退燒與集中風險的追蹤是 B 級背書[3][7]。
- **資本市場分歧本身就是訊號**:同一份 FY2025 財報下,多頭看「平台化+海外」、空頭看「單 IP+Q4 放緩」,市場「split」[3]——資金到位但對持久性下注未定。

## 5. 多時程 / Multi-horizon (T0–T3)
- **T0(現在)= 過熱**:Pop Mart 二級市場退燒(Labubu「Luck」自 ¥500+ 跌至 ~¥108、第三代整盒自高點 −80%)[7],股價年報後 −15%、自 8 月高點 −25%[3][7];情緒+估值擠在高位,échantillon 同溫層把「Labubu=結構」當定局。
- **T1(1–3y)= 結構**:平台基本面(海外 43.8%、多 IP 梯隊、Bandai/Sanrio/Disney 獲利)真實但大多已定價;贏家是「擁有可重複管線者」而非追單一角色。
- **T2(3–5y)= 結構**:海外滲透(美洲 +748%)、IP-to-everything(樂園/影視/卡牌)決定誰把情緒消費變成年金式現金流。
- **T3(5–10y)= 質變**:若「IP=注意力經濟的消費護城河」成立(綁 [[youth-culture-shifts]] 的注意力壟斷),擁有跨世代角色庫+造星機器者(Pop Mart、Bandai、Disney、Sanrio)將取得類奢侈品的定價權與毛利。

## 6. 爆發條件 + 里程碑階梯 / Milestone ladder
1. **Pop Mart 海外佔比過半** — 現 43.8%(FY2025)[2];驗證:看 FY2026 中報/年報海外%。✅ 已達 40%+ 門檻 / ⏳ 過 50%。Next check: 2026 H1 報(2026/08)。
2. **「下一隻 Labubu」成立** — THE MONSTERS 仍佔 38.1%;Twinkle Twinkle(RMB 2.06B)/CRYBABY(2.93B)需逼近 Labubu 量級[6]。驗證:單一新 IP 年營收破 RMB 5B 且 THE MONSTERS 佔比降至 <30%。⏳。Next check: FY2026 年報。
3. **二級市場價格止跌企穩** — Labubu 第三代自高點 −80%[7];驗證:StockX/得物熱門款價格連兩季不再破底。❌(仍在下行/築底)。Next check: 季度。
4. **Disney 樂園+DTC 雙引擎獲利拐點** — DTC 全年 +$1.3B、國際樂園營益 +25%[11];驗證:FY2026 DTC 營益再擴張 + 樂園營益正成長。⏳。Next check: Disney FY2026 季報。
5. **重啟片單命中率** — 2025 video-game 改編(Minecraft $958M 全球)強,但傳統 IP「不如以往硬」(Demon Slayer 勝 Superman)[16]。驗證:看 2026 主力續集/重啟全球票房中位數 vs 製作成本倍數。⏳。Next check: 半年票房盤點。
6. **Bandai Gundam 維持高原** — FY2026 ¥254.3B 創高[8];驗證:FY2027 是否守住 ¥200B+ 而非一次性峰值。⏳。Next check: Bandai FY2027 年報。

## 7. 時代影響與交互 / Era impact & interactions
情緒/IP 消費是 [[youth-culture-shifts]] 的「需求出海口」:當注意力被短影音/社群壟斷,**IP 成為注意力變現的容器**——Pop Mart 把 Xiaohongshu/TikTok 曬單流量直接導成盲盒復購,Twinkle Twinkle 近 10 億曝光即轉化為銷量[6]。這與奢侈品邏輯趨同(Time 直接以「Gen-Z luxury」框架報導)[15],與 [[luxury-and-apparel]] 的「身份/情緒溢價」共振。同時,IP 飛輪正吞噬通用內容:2025 票房顯示**有 IP 才有保底,但老 IP 邊際遞減**(video-game IP 強、Marvel/Star Wars 收縮)[16],對單片片廠是結構性逆風、對多 IP 平台(Disney/Bandai)是集中化紅利。

## 8. 同溫層 + 自我打臉 / Echo-chamber + self-refutation
- **同溫層 GAP**:消費/社群圈把「Labubu=Pop Mart 永動機」當定局,但硬數據是 THE MONSTERS 仍佔 38.1% 且 Q4 增速放緩、二級市場 −50~80%、股價自高點 −25%[3][7];真正的 falsifier 是「**下一個 IP 能否獨立扛量**」,目前未證。反向同溫層風險:把「盲盒=智商稅泡沫」一語否定,忽略 6 個 IP 破 20 億、海外 43.8% 的平台化硬證據[2][6]。
- **打臉我的多頭**:(a)「Pop Mart 已質變為平台」——但若 Labubu 退燒而 Twinkle Twinkle/CRYBABY 撐不起(FY2025 仍遠低於 THE MONSTERS)[6],平台敘事就是單 IP 週期的包裝;(b)「IP 是永久護城河」——Hasbro/Mattel/Disney 證明 IP 也會老化、命中率會掉(Barbie 紅利退潮、Marvel 收縮)[13][16],護城河需持續資本再投入而非躺收;(c)「海外=無限成長」——美洲 +748% 是低基期,關稅/文化退燒/仿冒(義烏白牌)是真實尾部風險。
- **空方核心**:盲盒在中國面臨未成年人保護/隨機消費監管尾部風險;情緒消費對景氣與「下一個爆款」高度敏感,本質是 hits-driven,估值卻按平台給;退燒一旦從二級市場傳導到一級復購,184.7% 的成長基期將極難 comp。
